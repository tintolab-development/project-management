사용 방법
  • 방법 1 : 엑셀에서 헤더 포함 행을 복사해서 오른쪽 칸에 그대로 붙여넣기
  • 방법 2 : 엑셀에서 CSV UTF-8로 저장 후 업로드
  • code 가 기존 항목과 같으면 업데이트, 없으면 신규 생성
  • 확정 항목은 보호를 위해 일괄업데이트에서 자동 건너뜀
지원 컬럼: code, type, domain, title, description, priority, status, owner, dueDate, clientResponse, finalConfirmedValue
한글 헤더도 지원: 코드, 유형, 도메인, 제목, 설명, 우선순위, 상태, 담당자, 마감일, 고객회신값, 최종확인값
상태값은 반드시 아래 3개 중 하나로만 사용: 논의 / 방향합의 / 확정
CSV 업로드
