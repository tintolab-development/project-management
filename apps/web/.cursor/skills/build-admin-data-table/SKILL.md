# Build Admin Data Table

## Instructions

1. Define the table owner slice.
2. Separate query/filter state from row rendering.
3. Use server state via TanStack Query.
4. Keep sorting, filters, pagination, and selection explicit.
5. Use shadcn/ui table-related primitives and surrounding controls.
6. Handle loading, empty, error, and refetch states explicitly.
7. Keep bulk actions and row actions isolated from base table structure.
