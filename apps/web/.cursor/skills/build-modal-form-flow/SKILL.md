# Build Modal Form Flow

## Instructions

1. Choose dialog, drawer, or sheet based on task weight and screen context.
2. Define zod schema first.
3. Wire react-hook-form with resolver.
4. Use shadcn/ui structure for title, description, body, actions, and errors.
5. Handle submit pending state and optimistic UX carefully.
6. Preserve focus management and keyboard behavior.
7. Invalidate only the necessary queries after success.
